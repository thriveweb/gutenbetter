== Description ==

This plugin was created to enhance your Gutenberg experience. 

Features include...

+ Additional margins around blocks for better editing
+ Enable/disable the block editor for specific post types
+ Dynamically resize the block editor sidebar
+ Hide active blocks from the frontend while editing
+ Force preview mode for ACF blocks when editor is loaded
+ Remove block directory from block search
